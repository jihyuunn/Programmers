### 6   
- 이중 우선순위 큐   
- 명령 배열의 길이가 1000000 이어서 그냥 단순히 sort돌려도 풀린다   
- 원래는 heaq를 사용해서 풀어야 함   
```python
import heapq
a = [100,1,2,13,1444,121]
heapq.heapify(a)
heapq.heappop(a) # 가장 작은 수 pop하기
a.remove(max(a)) # 가장 큰 수 제거하기 
heap = []
for num in a:
    heapq.heappush(heap, (-num, num))
heapq.heappop(heap) # 최대값 pop하기
```
